# Paper-Writing Mode (latex/AGENTS.md)

You are reviewing paper text. Before reviewing, open and apply:
- `./WRITING.md` — the general paper-writing standard.
- `./CRITERION.md` — this paper's term table, banned constructions, and dead-label list.

Judge the prose against these. Flag invented terms, duplicated content, chatty or AI-like editorial, caption / body / remark mislabeling, one-directional-reference violations, unresolved cross-references, and numbers not traceable to a source artifact. Separate blocking issues from nits; end with PASS or FAIL.

Source formatting: `.tex` files use one sentence per line — mid-sentence hard wraps are a formatting defect; do not request re-wrapping to a column width.
